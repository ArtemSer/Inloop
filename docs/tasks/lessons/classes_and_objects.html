Inloop / Tasks / Classes and objects

In this task it is your job to implement a Taxi class that represents taxis which have a driver and seats for four passengers. Goal of this task is to give you an impression on how classes and methods in Java work and how to use them.

To solve this task follow the instructions given in the task description.

Task 1

Every passenger is represented by an instance of the class Human that has the String attributes forename and name (the surname). Every Human can be asked for his name using getName() or his forename using getForename().

Task 2

Every class in Java is derived from the basic class Object which implements a method called toString(). This method provides the String representation of a certain object of the class. It is often used implicitly in Java like in this case: 

System.out.println(new Human("Udo", "G"));
